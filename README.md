# Aether-Live-Weather-Intelligence-Dashboard
Here’s a concise GitHub description under 350 characters:  **Aether — Live Weather Intelligence Dashboard** is a modern weather app with real-time data, city search, current conditions, stats (humidity, wind, pressure), UV/sun info, wind compass, hourly charts, and 5-day forecast. Built with HTML, CSS, JavaScript, Bootstrap, and OpenWeatherMap API.
